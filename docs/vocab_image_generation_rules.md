# vocab1300 画像生成ルール

**対象**: 青のまなびクエスト — 語彙力UP1300  
**作成日**: 2026-07-02

---

## 1. 共通ルール

すべての画像に適用する基本方針。

| 項目 | 内容 |
|---|---|
| 出典 | 教材画像は使わない。生成AIで作るオリジナル画像のみ使用する |
| 背景 | 白背景 |
| スタイル | 白黒のシンプルな手描き風線画 |
| 対象年齢 | 小学生向け教材イラスト |
| サイズ・比率 | 正方形 |
| 文字・記号 | 文字・数字・記号は入れない |
| 装飾 | 枠線・吹き出し・ラベルは入れない |
| 構図 | 主役を大きく描く。余計な背景を入れない |
| 視認性 | アプリ内で小さく表示しても何が描かれているか分かる絵にする |

## 2. subtype別の描き方方針

### counter_image（数え方・助数詞）

- 数え方（助数詞）に対応するモノを主役に描く
- 「1つ」「1本」のように、数量が分かる描き方にする
- モノの形・特徴が一目で分かるシルエットにする
- 同じ助数詞でも対象が異なる語がある（例: 一丁 → そろばん / 豆腐）ため、モノ自体を明確に描く
- 人物は原則不要。モノだけで成立する構図にする

### manner_word（礼儀・作法・しぐさ）

- 人が礼儀・作法・しぐさをしているシーンを描く
- 動作の「何をやっているか」が分かるシンプルな人物イラスト
- 人物は2人以上のことが多い（相手がいる場面）
- 動作の方向・関係性が伝わる構図にする
- 表情ではなく動作・ポーズで内容を表現する

### clothing_word（和服・衣服の部位）

- 和服・衣服を着た人物を描き、注目部位を分かりやすく示す
- 全体と注目部位の両方が伝わる構図にする
- 部位の形・位置が視覚的に理解できるようにする
- 似た語（留め袖・振り袖など）は、袖の長さ・全体の雰囲気など違いが分かるように描く
- 裾・袂・懐など部位名の場合は、対象部分を大きく・はっきり示す構図にする

### weather_word（天気・気象・季節の風）

**注意: 30問あり、似た語が多い。混同しないよう特に区別を意識する。**

#### 雨・水分系（p38 / p50）

- 雨の強さ・降り方・季節感が区別できるようにする
  - 篠突く雨: 縦に細い線が密集して激しく降っている
  - にわか雨: 急な雨、晴れ間との対比で表現
  - 夕立: 夏、激しい雨と暗い空
  - 通り雨: ざっと降ってすぐやむ、軽い雨
  - 菜種梅雨: 春、花（菜の花）と細かい雨
  - 空梅雨: 梅雨なのに晴れた空（雨なし）
  - しぐれ: 秋冬の端境期、軽くぱらぱら
  - 五月雨: 梅雨のしとしと長雨
  - 小糠雨: 霧のような極細の雨
  - 狐の嫁入り: 晴れているのに雨（太陽と雨の同時描写）
  - 春雨: 春のしとしと雨
  - 入梅: 梅雨の始まり（初めて傘を差すシーンなど）

#### 風系（p62 / p74）

- 風の向き・強さ・季節感が区別できるようにする
  - はやて: 突然の強風、人が驚いている
  - 野分け: 秋冬の荒々しい風、草木が大きくなびく
  - 春一番: 暖かい南風、春らしさ
  - 山背: 東北の海側に吹く冷たい風（山と海のシルエット）
  - おろし: 山から下へ吹き下ろす風。山の斜面に沿って、木・草・服・髪が下向きになびく様子で表現する
  - 木枯らし: 秋冬の冷たい強風、枯れ葉が舞う
  - つむじ風: うずまき状の風
  - 空っ風: 冷たく乾いた北風、雲なし
  - 花風: 桜の花びらを揺らす風
  - 追い風: 人の進行方向と同じ向きの風
  - 青嵐: 初夏の若葉を揺らす強い風
  - 向かい風: 人の進行方向と逆向きの風

#### 雲・霧・霞系（p86）

- 雲・霧・霞・靄は、空や見え方の違いが分かるようにする
  - 入道雲: 夏、もくもくと高く盛り上がった大きな積乱雲
  - 霞: 春、遠くがぼんやり見える（遠景がぼやけた景色）
  - 霧: 地面近くが白くなり、視界が悪い（近距離の白いもや）
  - ひつじ雲: 小さな丸い雲が群れで並んでいる
  - 靄: 空気全体がぼんやり（霧より薄いもや）
  - いわし雲: 魚のうろこ状の小さな雲が広がっている

### day_time_word（一日の時間帯）

- 空・地平線・建物などで時間帯を表現する
- 明るさ・太陽や月の位置・星の有無で区別する
  - あけぼの: 夜明け前後、空の端がほんのり明るい
  - 未明: 夜中を過ぎたが、まだ暗い
  - 昼下がり: 明るい昼の空、太陽が少し西側
  - たそがれ: 夕方、オレンジ〜暗い空
  - 宵の口: 夕暮れ直後、日が沈んだばかり
  - 夜半: 深夜、暗く静かな夜
- 1枚でも明暗の違いが分かる構図にする（空のグラデーション、星・月・太陽の有無）

### body_part_word（体の部位）

- 体のシルエット全体の中で、対象部位を大きく・明確に示す
- 矢印やマーカーは使わず、対象部位を拡大・強調した描き方にする
- 位置が分かりにくい部位（きびす、土ふまず）は足のシルエットの中で該当箇所を際立たせる
- 正面・側面・背面など、対象部位が最も分かりやすいアングルを選ぶ

### house_part_word（日本家屋の部位・建具）

- 日本家屋の外観または室内を描き、注目部分が分かるようにする
- 似た語は区別を明確にする
  - 鴨居（上の溝のある横木）と敷居（下の溝のある横木）は上下の位置が分かる構図
  - 軒（屋根の張り出し）とひさし（玄関・窓上の小屋根）は外観で区別
  - ふすま（布張りの引き戸）と障子（白い紙張りの引き戸）は素材感・模様で区別
- 建具（障子・ふすま）は、開閉している様子より「どんな見た目か」を中心に描く

### cooking_word（料理の動作）

- 調理動作を主役にする
- 食材・調理器具・手の動きで「何をしているか」を表現する
- 似た動作を区別する
  - あえる（野菜と調味料を和える）・からめる（ソースをまとわせる）・まぶす（表面に粉をつける）は動作とターゲットの違いで区別
  - 湯がく（さっと熱湯に通す）・ふかす（蒸気で蒸す）・煮る（水と一緒に火にかける）は調理方法で区別
- 動作が完了した状態ではなく、動作の最中を描く

### old_life_tool_word（昔の生活道具）

- 道具の形と使い方の両方が伝わるイラストにする
- 現代の子どもが見て「何に使うものか」が想像できる描き方にする
  - 火鉢: 器の中に炭、上に手をかざす
  - おひつ: 木製の丸い容器（フタあり）
  - 縁台: 屋外に置く細長い腰かけ
  - すだれ: 竹の板が並んで吊り下がる、日よけ
  - 蚊帳: 寝床を覆うテント状の薄い網
  - いろり: 床を四角く切り込んだ炉、上に鍋をつるす
- 道具単体だけでなく、使われている場面（人が使っているシーン）を入れてもよい

## 3. 画像ファイル名ルール

## 4. 画像生成フロー（予定）

1. `image-prompts/vocab1300_image_targets_v1.json` で対象語句を確認
2. 各語句に対してプロンプトを作成（`image-prompts/` 配下に保存）
3. 生成AIで画像生成 → `generated-images/` に一時保存
4. 採用画像を `app/images/` に移動・リネーム
5. 問題JSONの `image` パスと照合して完了確認
